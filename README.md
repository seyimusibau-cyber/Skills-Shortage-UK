# Skills Shortage UK

This repository is dedicated to analyzing and providing strategic solutions for the technology skills shortage, with a specific focus on the North East of the United Kingdom.

## Projects Included

### [UK_NE_Tech_Skills_Dashboard](./UK_NE_Tech_Skills_Dashboard/)
An interactive Streamlit dashboard and exploratory data analysis project that visualizes the current vs. required tech workforce in the region. 

It highlights key roles such as Software Engineering, AI, and Cybersecurity, and provides **10 Actionable Strategies** to manage the skills shortage effectively based on industry best practices and regional initiatives.

- **Data Insights:** Visualize shortfall percentages, days-to-hire, and skills gaps across 10 critical tech roles.
- **Strategic Solutions:** Read through 10 ways to tackle the shortage, from internal upskilling to engaging with local educational institutions.

### Getting Started

To explore the dashboard locally, navigate to the dashboard directory:

```bash
cd UK_NE_Tech_Skills_Dashboard
pip install -r requirements.txt
streamlit run app.py
```
